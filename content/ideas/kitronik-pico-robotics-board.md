## Pico Simply Robotics Board

| Attribute | Value |
|:-------|:---------|
| Supply Voltage | 3 - 12V |
| Number of Servo Channels | 8 |
| Number of Motor Channels | 4 DC / 2 Stepper |
| Servo / Motor Voltage | Same as Supply Voltage |
| Max Continuous Current (all Servos and Motors) | 10 Amps |
| Max Motor Current | 1A / Motor |
| Additional IO | 5 (via solder pads) |
| Additional GND | 3 (via solder pads) |
| Additional 3V3 | 3 (via solder pads) |
