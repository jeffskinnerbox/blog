### MS Windows Product Key
For a computer with Windows 11 pre-installed,
the product key is embedded in the motherboard firmware rather than on a sticker, and it usually activates automatically.
You can retrieve this key by running a command in the Command Prompt or using the ShowKeyPlus app.

* [How to Find Your Windows 11 Product Key](https://www.youtube.com/watch?v=hsE-IcU3Y_I&t=1s)
